# File Organizer CLI

A simple command-line tool to organize files in a directory by type.

## Features
- Categorizes files into Images, Documents, Code, and Others
- Automatically creates folders
- Works on any given directory

## Usage
python organizer.py <folder_path>

## Example
python organizer.py ./test_folder
python organizer.py <folder_path>

## Undo organization:
python undo.py <folder_path>
